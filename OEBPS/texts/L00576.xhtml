[handschriftlich Hugo von Hofmannsthal:] Sie hat ihm [handschriftlich Felix Salten:] (mir) [handschriftlich Hugo von Hofmannsthal:] recht gut [handschriftlich Felix Salten:] (sehr gut! das »recht gut« ist nur [handschriftlich Hugo von Hofmannsthal:] meine [handschriftlich Felix Salten:] ((seine)) Bescheidenheit) gefallen. [handschriftlich Hugo von Hofmannsthal:] Natürlich ist er [handschriftlich Felix Salten:] (ich) [handschriftlich Hugo von Hofmannsthal:] sofort wieder abgereist. [handschriftlich Felix Salten:] Das hat er [handschriftlich Hugo von Hofmannsthal:] (habe ich) [handschriftlich Felix Salten:] seit sechs Wochen vorher gewusst. Dies wünscht Euch
